# Evidence Runtime

Evidence represents validated context. Evidence is not truth.

## Evidence Contains

- Origin.
- Confidence.
- Freshness.
- Supporting observations.
- Conflicting observations.
- Expiration.
- Provenance.
- Relationships.
- Trace identifiers.

## Behavior

Evidence can age, decay, become stale, merge, split, and contradict. Phase 16 includes creation and decay foundations; future phases may expand merge/split policies.

## Forbidden Uses

Evidence must never become gameplay, movement, navigation, execution, or presentation.
