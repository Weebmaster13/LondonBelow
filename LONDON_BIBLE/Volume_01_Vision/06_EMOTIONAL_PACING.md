# Emotional Pacing

London Below should breathe.

Fear requires contrast. The game needs warmth, quiet, anticipation, pressure, release, sadness, and dread. Constant loud pressure makes players numb.

## Pacing Shape

1. Safety: family and home feel real.
2. First wrongness: a small detail fails to match memory.
3. Denial: the player tries to continue normally.
4. Entry: the Building accepts the player.
5. Pressure: identity and memory begin to erode.
6. Revelation: a puzzle or memory changes meaning.
7. Escape attempt: progress feels possible.
8. Cost: success requires loss or recognition.

## Director Use

Horror, Environment, Lighting, Audio, and future Monster Directors should coordinate pressure so the experience escalates with intention.

## Replay Pacing

Replay should not simply be harder. It should make early comfort suspicious and late horror more tragic.

## Open Design Questions

- Which chapter should deliver the first irreversible identity loss?
- How often should the game offer true emotional rest?
