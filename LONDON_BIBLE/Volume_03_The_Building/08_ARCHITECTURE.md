# Building Architecture

Architecture is one of the Building's languages.

## Visual Direction

London Below should use Victorian London atmosphere, foggy streets, old doors, tall interiors, narrow passages, staircases, gaslight, rain, and impossible underground structures without copying existing maps or layouts.

## Spatial Meaning

Architecture should communicate:

- Safety becoming unreliable.
- Childhood scale.
- Memory compression.
- Emotional thresholds.
- The Building's ancient appetite.

## Engine Connection

World Intelligence should classify districts, rooms, micro-zones, safe rooms, puzzle rooms, chase routes, and affordances. Directors should respect those classifications.

## Open Design Questions

- What is the signature silhouette of the Building?
- Which architectural motifs belong only to replay revelations?
