# The Building

The Building is alive, ancient, and older than the main entity.

## Canon Identity

The Building is not a normal haunted location. It is an intelligence that consumes identity, memories, and emotional connections. It may appear as a building, a set of buildings, corridors below London, rooms from memory, or architecture that should not fit together.

## Relationship To The Entity

The main entity did not create the Building. The entity uses it.

This distinction matters. The Building is not merely the monster's house. It has its own perspective, appetite, memory, and rules.

## Design Use

The Building can:

- Reframe rooms.
- Preserve emotional residue.
- Hide or reveal memory fragments.
- Make safe spaces uncertain.
- Tell truth from its own perspective.

## What It Does Not Own

The Building does not automatically control every scare, monster behavior, or puzzle. Directors and gameplay systems must still coordinate through the London Engine architecture.

## Open Design Questions

- Is the Building a single place, a repeating structure, or a network beneath London?
- What did the Building consume before the player arrived?
