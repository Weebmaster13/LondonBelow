# Entity Philosophy

Entities in London Below are expressions of meaning, not generic enemies.

## Core Rule

Every entity should reveal something about the Building, identity, family, memory, or emotional pressure.

## Main Entity

The main entity did not create the Building. It uses the Building and should feel intelligent, patient, and cruelly curious.

## Smaller Entities

Future crawler creatures may alert the main entity, harass players, mislead, or create pressure. They should not replace the main entity's role.

## Boundaries

Do not implement Monster AI in Bible files. Do not finalize entity models, names, animations, or chase logic here.

## Open Design Questions

- What does each entity want from the player?
- Which entities are literal and which are Building-shaped symbols?
