# Thought Runtime

Thoughts are promoted hypotheses.

## Thought Lifecycle

Thoughts can be born, become dormant, compete, reinforce, contradict, decay, merge, split, archive, and resurrect. Each transition is recorded.

## Boundary

A thought is not a movement, attack, goal, animation, or execution request. It is a reasoning candidate.

## Validation

Invalid thought transitions reject. Archived thoughts can only resurrect.
