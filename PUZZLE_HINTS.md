# Puzzle Hints

Puzzle hints are progressive, delayed, server-approved presentation hooks.

Hints should be observation-driven, co-op-aware, accessible, and fair. They must never instantly reveal the answer by default. The hint runtime uses a delay before showing the next configured hint.

Future Puzzle Director integration may decide whether pressure, silence, lighting, audio, or environmental support should accompany a hint. The hint runtime does not execute those effects.
