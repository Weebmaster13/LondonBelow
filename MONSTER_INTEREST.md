# Monster Interest

Interest is the foundation for monster attention.

## Inputs

Potential future inputs include noise, movement, light, identity exposure, memory fragments, doors, Journal activity, player hesitation, group separation, and recent objectives.

## Rules

- Interest is bounded from 0 to 100.
- Interest decays.
- Interest must never explode infinitely.
- Interest is a reason for attention, not permission to attack.

## Intent Relationship

MonsterMind combines interest with curiosity, patience, threat, territory, and search scores. The chosen intent includes reasons so diagnostics can explain why it happened.

## Future Use

Observation Engine should feed trusted facts. Client claims must never directly create monster interest truth.
